# DARKSOULx69-png2ico (Basic) by DARKSOULx69

PyQt5 based simple **.png** to **.ico** converter.

## Installation
- Install Python
- Install requirements
  - **Manually install**  

    ```
    $ pip install PyQt5
    $ pip install pillow
  - **Automatically install**  

    ```
    $ pip install -r requirements.txt
    ````
## Usage

Open cmd  
Browse png2ico folder
Type **"Python main.py"**  
or run **"run.bat"**
